# public/audio/music

Drop-in folder for professionally produced **original** music tracks.

The game works with no files here — it runs an original real-time generative
soundtrack instead. To add produced tracks, place broadly compatible audio
(`.mp3`, `.m4a`, `.ogg`/`.opus`) here and register them as described in
[`../../../MUSIC_PRODUCTION.md`](../../../MUSIC_PRODUCTION.md).

Only add music you created or that is explicitly licensed for redistribution,
and record it in `ASSET_LICENSES.md`. Never add copyrighted music.
